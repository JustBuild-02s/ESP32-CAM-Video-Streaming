# ESP32-CAM Live Video Streaming 📷

This project demonstrates live video streaming using the ESP32-CAM module and Arduino IDE.

## 📌 Project Overview

The ESP32-CAM module is used to capture and stream live video over WiFi.
The device hosts a web server that allows users to view the camera feed directly from a browser.

## ⚙️ Hardware Used

* ESP32-CAM (AI Thinker)
* ESP32-CAM-MB Programmer
* OV2640 Camera Module
* USB Cable

## 🛠 Software

* Arduino IDE
* ESP32 Board Package

## 📡 Features

* Live video streaming over WiFi
* Web-based camera control
* Adjustable frame size and resolution
* Low-cost IoT camera solution

## 🔧 Setup Instructions

1. Install ESP32 board in Arduino IDE
2. Select **AI Thinker ESP32-CAM**
3. Upload **CameraWebServer example**
4. Open Serial Monitor
5. Copy the IP address shown
6. Open the IP in your browser

## 📷 Output

The ESP32 hosts a local web server where the camera stream can be viewed in real time.
